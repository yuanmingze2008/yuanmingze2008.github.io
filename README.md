# Yummz Academic Homepage

This repository hosts the GitHub Pages site for `yuanmingze2008.github.io`.

The site is based on the Academic Pages template and is currently configured as a lightweight academic-style homepage for:

- learning notes
- deep learning
- agent systems
- small research-oriented projects

Main pages:

- `/`: English about page
- `/zh/`: Chinese about page
- `/notes/`: Chinese learning notes
- `/portfolio/`: projects
- `/cv/`: concise CV
